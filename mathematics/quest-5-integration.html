Side Quest 5.1: Basic Integrals

Integration rules

Historical Context

Isaac Newton & Gottfried Leibniz (17th century) - Independently developed integration as the inverse of differentiation.

Bernhard Riemann (19th century) - Developed rigorous definition of definite integrals.

Henri Lebesgue (20th century) - Advanced integration theory with measure theory.

Integration Rules

∫xⁿ dx = xⁿ⁺¹/(n+1) + C

∫eˣ dx = eˣ + C

∫1/x dx = ln|x| + C

∫cos x dx = sin x + C

∫sin x dx = -cos x + C

Real-World Applications

Physics - Calculating work, energy, and area under curves
Engineering - Volume calculations and structural analysis
Economics - Consumer surplus and total revenue

Practice Exercise

Problem: Evaluate ∫(3x² + 2x + 1) dx

Solution: Integrate term by term:

∫3x² dx = 3×(x³/3) = x³

∫2x dx = 2×(x²/2) = x²

∫1 dx = x

Answer: x³ + x² + x + C

Common Mistakes & Tips

Mistake: Forgetting the constant of integration (+C)

Tip: Always include +C for indefinite integrals

Mistake: Misapplying power rule to 1/x

Tip: ∫1/x dx = ln|x| + C, not x⁰/0
